# Day 7 — LLM Limitations

## Bias
Models reflect training data biases.

## Drift
- Data drift: world changes
- Concept drift: meaning changes

## Failure Modes
- Hallucination
- Overgeneralization
- False confidence
- Instruction conflict

## Prompt Overfitting
Prompts tuned to examples fail on new inputs.

## Distribution Shift
Training data ≠ real usage data.
Causes performance drop.

Key Rule:
LLMs sound confident even when wrong.
Never trust without verification.

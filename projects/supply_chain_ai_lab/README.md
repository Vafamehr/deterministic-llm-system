# Supply Chain AI Lab

A long-term project for building realistic supply chain AI systems with a modular architecture.

This lab is designed to serve three goals:

- learn advanced data science, ML, and LLM system design
- build strong portfolio projects for GitHub and job applications
- prepare for supply chain, ML, and AI system design interviews

The project will evolve gradually and focus on real supply chain problems such as:

- demand forecasting
- inventory and replenishment
- allocation and transfers
- disruption management
- assortment optimization
- supplier risk
- recommendation systems

The architecture will emphasize:

- modular design
- simulation-based evaluation
- deterministic decision layers
- bounded LLM and agent integration
- documentation for learning and interviews
